# Markdown 基础语法

## 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 文本样式

| 效果 | 语法 |
|------|------|
| **加粗** | `**文本**` 或 `__文本__` |
| *斜体* | `*文本*` 或 `_文本_` |
| ***粗斜体*** | `***文本***` |
| ~~删除线~~ | `~~文本~~` |
| ==高亮== | `==文本==` |
| 行内代码 | `` `代码` `` |

## 列表

### 无序列表

```markdown
- 项目一
- 项目二
  - 嵌套项目
  - 嵌套项目
```

### 有序列表

```markdown
1. 第一项
2. 第二项
   1. 嵌套第一项
   2. 嵌套第二项
```

### 任务列表

```markdown
- [x] 已完成任务
- [ ] 未完成任务
- [ ] 待办事项
```

## 引用

```markdown
> 这是一个引用
>
> 引用可以有多行
>> 嵌套引用
```

## 代码块

```markdown
```python
print("Hello World!")
```

```javascript
const name = "Obsidian";
```
```

## 链接

### 普通链接

```markdown
[链接文本](https://example.com)
[链接文本](https://example.com "标题")
```

### 锚点链接

```markdown
[跳转到标题](#标题名称)
```

# Obsidian 特有语法

## 双向链接

### 基础链接

```markdown
[[笔记名称]]
[[笔记名称|显示文本]]
```

### 链接到标题

```markdown
[[笔记名称#标题名称]]
[[笔记名称#标题名称|显示文本]]
```

### 链接到块

```markdown
[[笔记名称^块引用ID]]
```

## 标签

```markdown
#标签名称
#parent/child  (层级标签)
```

## 嵌入

### 嵌入笔记

```markdown
![[笔记名称]]
![[笔记名称#标题]]
```

### 嵌入图片

```markdown
![图片描述](assets/image.png)
![[image.png]]  (Obsidian 内部)
```

### 嵌入其他文件

```markdown
![[file.pdf]]
![[file.csv]]
```

## 块引用

在任意段落末尾添加 `^` 和自定义 ID：

```markdown
这是一段可以被引用的文本。^block-id
```

引用该块：

```markdown
[[笔记名称^block-id]]
```

## 数学公式

Obsidian 使用 LaTeX 语法：

```markdown
$$
E = mc^2
$$

行内公式：$E = mc^2$
```

## 表格

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |
```

## 分隔线

```markdown
---
***
```

## 脚注

```markdown
这是一个带脚注的句子[^1]。

[^1]: 这是脚注的内容。
```

# Obsidian 快捷键

## 文件操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + O` | 切换或创建笔记 |
| `Ctrl + P` | 快速打开命令面板 |
| `Ctrl + N` | 创建新笔记 |
| `Ctrl + S` | 保存笔记 |
| `Ctrl + Shift + S` | 另存为 |

## 编辑操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + B` | 加粗 |
| `Ctrl + I` | 斜体 |
| `Ctrl + K` | 插入链接 |
| `Ctrl + L` | 选中当前行 |
| `Ctrl + D` | 选中相同词 |

## 导航

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Tab` | 切换标签页 |
| `Ctrl + Shift + Tab` | 反向切换标签页 |
| `Ctrl + F` | 查找 |
| `Ctrl + G` | 跳转到行 |

## 视图

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + E` | 打开侧边栏 |
| `Ctrl + \` | 切换侧边栏 |
| `Ctrl + Shift + P` | 命令面板 |
