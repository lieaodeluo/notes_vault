## 二、文本样式

| 语法 | 效果 |
|------|------|
| `**加粗文本**` | **加粗文本** |
| `__加粗文本__` | __加粗文本__ |
| `*斜体文本*` | *斜体文本* |
| `_斜体文本_` | _斜体文本_ |

| `~~删除线~~` | ~~删除线~~ |
| `==高亮==` | ==高亮== |
| `` `行内代码` `` | `行内代码` |

## 三、列表

### 3.1 无序列表

**语法：**
```markdown
- 项目一
- 项目二
  - 嵌套项目A
  - 嵌套项目B
- 项目三
```

**效果：**
- 项目一
- 项目二
  - 嵌套项目A
  - 嵌套项目B
- 项目三

### 3.2 有序列表

**语法：**
```markdown
1. 第一项
2. 第二项
   1. 嵌套第一项
   2. 嵌套第二项
3. 第三项
```

**效果：**
1. 第一项
2. 第二项
   1. 嵌套第一项
   2. 嵌套第二项
3. 第三项

### 3.3 任务列表

**语法：**
```markdown
- [x] 已完成任务
- [ ] 未完成任务
- [ ] 待办事项
```

**效果：**
- [x] 已完成任务
- [ ] 未完成任务
- [ ] 待办事项

# Obsidian 特有语法

## 一、双向链接

| 语法 | 效果说明 |
|------|----------|
| `[[笔记名称]]` | 创建指向"笔记名称"的双向链接 |
| `[[笔记名称\|显示文本]]` | 创建带自定义显示文本的双向链接 |
| `[[笔记名称#标题]]` | 链接到笔记中的特定标题 |
| `[[笔记名称^块ID]]` | 链接到笔记中的特定块 |

---

## 二、嵌入

| 语法 | 效果说明 |
|------|----------|
| `![[笔记名称]]` | 在当前笔记中嵌入另一篇笔记 |
| `![[笔记名称#标题]]` | 嵌入笔记中的特定标题部分 |
| `![[image.png]]` | 嵌入图片文件 |
| `![[file.pdf]]` | 嵌入PDF文件 |

---

## 三、标签

**语法：**
```markdown
#标签名称
#parent/child
```

**效果：**
#学习 #工作/会议

---

## 四、块引用

**语法：**
```markdown
这是一段可以被引用的文本。^block-id
```

**效果：**
这是一段可以被引用的文本。^block-id

引用方式：`[[笔记名称^block-id]]`

---

## 五、数学公式

**语法：**
```markdown
$$E = mc^2$$
行内公式：$E = mc^2$
```

**效果：**
$$E = mc^2$$
行内公式：$E = mc^2$

---

## 六、脚注

**语法：**
```markdown
这是带脚注的句子[^1]。

[^1]: 这是脚注内容。
```

**效果：**
这是带脚注的句子[^1]。

[^1]: 这是脚注内容。

# Obsidian 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + O` | 切换或创建笔记 |
| `Ctrl + P` | 快速打开命令面板 |
| `Ctrl + N` | 创建新笔记 |
| `Ctrl + S` | 保存笔记 |
| `Ctrl + B` | 加粗文本 |
| `Ctrl + I` | 斜体文本 |
| `Ctrl + K` | 插入链接 |
| `Ctrl + F` | 查找 |
| `Ctrl + E` | 打开侧边栏 |
